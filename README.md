# Farm Tokenization
A decentralized platform for tokenizing farmland and enabling fractional ownership through shares.

## 🚀 Features

- Register farms with detailed information
- Purchase and transfer farm shares
- Verification system for farm authenticity
- Investor portfolio tracking
- Secure ownership management

## 🔒 Security Features

- Owner-only access control for farm verification
- Input validation for all numerical values
- Share transfer validation
- Protected investor records

## 🛠 Technical Stack

- Clarity Smart Contracts
- Stacks Blockchain
- Clarinet for testing

## 📋 Usage Instructions

1. Install Clarinet
```bash
curl -L https://github.com/hirosystems/clarinet/releases/download/v1.0.0/clarinet-linux-x64.tar.gz | tar xz
```

2. Initialize project
```bash
clarinet new farm-tokenization && cd farm-tokenization
```

3. Deploy contract
```bash
clarinet deploy
```

4. Run tests
```bash
clarinet test
```
